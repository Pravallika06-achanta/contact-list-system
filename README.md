# Contact List Management System

## 📌 Description
A simple Contact List Management System built using Python. This project allows users to store and manage contacts using file handling.

## ✨ Features
- Add Contact
- View Contacts
- Search Contact
- Delete Contact
- Delete All Contacts
- Prevent Empty Inputs
- Prevent Duplicate Contacts
- Store Contacts Permanently using `contacts.txt`

## 🛠️ Technologies Used
- Python
- File Handling
- Lists
- Loops
- Conditional Statements

## ▶️ How to Run

```bash
python main.py
```
